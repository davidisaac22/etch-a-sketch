# Etch-a-Sketch

This is a mini project from [The Odin Project](https://www.theodinproject.com/) curriculum, specifically part of the JavaScript course. It's a web-based recreation of the classic Etch-a-Sketch toy, allowing users to draw on a grid by hovering over squares.

## Features

- Interactive grid where you can draw by hovering over the squares
- Reset button to clear the drawing
- Mode selector to switch between drawing modes (e.g., solid color or random colors)
- Responsive design with a playful, toy-like UI

## How to Use

1. Open `index.html` in your web browser.
2. Hover over the grid squares to draw.
3. Use the dropdown to select different drawing modes.
4. Click the "Reset" button to clear the grid.

## Technologies Used

- HTML
- CSS
- JavaScript

## Project Structure

- `index.html`: The main HTML file with embedded CSS and structure.
- `index.js`: JavaScript file handling the drawing logic and interactions.

Enjoy sketching!
